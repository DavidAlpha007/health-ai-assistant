# 动图制作需求清单

## 一、通用制作规范

### 1. 分辨率要求
- 手机屏幕适配：1080px × 1920px（16:9 比例）
- 动图格式：MP4 或 GIF
- 帧率：24fps（确保动作流畅）

### 2. 内容要求
- 背景：纯黑色或透明背景（优先透明背景）
- 主体：工位真人实操动作
- 服装：简约职业装
- 背景元素：无多余装饰，仅保留必要的工位元素（如椅子、桌子）

### 3. 时长要求
- 肩颈酸痛场景：3分钟（3步动作，每步1分钟）
- 用眼过度场景：2分钟（4步动作，每步30秒）
- 久坐腰背僵硬场景：4分钟（5步动作，每步48秒）

### 4. 动作要求
- 动作清晰可见，突出核心缓解部位
- 无器械，可在工位执行
- 动作幅度适中，适合办公环境

## 二、各场景专属需求

### 1. 肩颈酸痛场景
- 动作数量：3步
- 核心动作：
  - 颈部左右拉伸
  - 肩部环绕运动
  - 斜方肌拉伸
- 语音引导：3分钟真人语音引导

### 2. 用眼过度场景
- 动作数量：4步
- 核心动作：
  - 眨眼运动
  - 眼球转动（上下左右）
  - 远眺放松
  - 眼周按摩
- 语音引导：2分钟真人语音引导

### 3. 久坐腰背僵硬场景
- 动作数量：5步
- 核心动作：
  - 坐姿转体
  - 脊柱伸展
  - 腰部扭转
  - 臀部拉伸
  - 肩部后缩
- 语音引导：4分钟真人语音引导

## 三、交付要求
- 交付格式：压缩包（包含所有动图和对应的语音文件）
- 命名规范：场景名称_动作序号（如：肩颈拉伸_01.mp4）
- 交付时间：第2个工作日结束前

## 四、验收标准
- 产品经理验收：确认动作清晰、无多余背景、适配手机屏幕
- 前端团队验收：确认动图格式和分辨率适合集成
